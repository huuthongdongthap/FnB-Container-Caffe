### SCREEN 06 — LOGO USAGE RULES (Do's & Don'ts)

Frame 1440×1080, nền `#0A0A0A`.

## HƯỚNG DẪN SỬ DỤNG

### Thứ tự paste vào Pencil.dev Desktop:
1. Mở file `fnb-ui.pen` trong Pencil.dev
2. Ghim `ARCHITECTURE.md` + `brand-thi-cong-specs.md`
3. Gõ `/style webapp-01-elegantluxury_light` để load style guide
4. Paste từng Screen **tuần tự** — đợi hoàn thành rồi paste screen tiếp
5. Sau mỗi 2-3 screens, dùng `get_screenshot` verify

### Style nhất quán:
- **Nền**: `#0A0A0A` (page), `#0F0F0F` (cards)
- **Border**: stroke `#1F1F1F` 1px
- **Gold accent**: `#C9A962` (primary), `#C9A200` (signage), `#FFD700` (neon)
- **Text**: `#FAF8F5` (primary), `#888888` (secondary), `#666666` (tertiary)
- **Fonts**: Playfair Display (display), Manrope (body), JetBrains Mono (code)
- **Corner radius**: 0px toàn bộ (luxury aesthetic)

### File tham chiếu:
- `designs/ARCHITECTURE.md` — Design tokens, layout, component specs
- `designs/brand-research.md` — 10 FnB brand benchmarks
- `designs/brand-thi-cong-specs.md` — Signage dims, materials, CMYK/Pantone, mockups
