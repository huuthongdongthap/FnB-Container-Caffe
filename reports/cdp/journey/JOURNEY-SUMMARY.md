## Top 3 Dropoffs to Fix (Highest ROI)

### #1: First Visit → Loyalty Signup (60% loss)
**Root cause**: Staff not offering signup consistently
**Fix**: Staff script training + 5K₫ bonus/signup (Week 1)
**Expected impact**: +20-40% signup rate → +60-120 members/month
**Cost**: 200K₫/month in bonuses

### #2: Walk-by → Enter Store (96.7% loss)
**Root cause**: No visible menu/WiFi info from street
**Fix**: A-frame sign with prices + WiFi/outlet info
**Expected impact**: +1-2% walk-in → +150-300 visits/month
**Cost**: 200K₫ one-time (print)

### #3: Signup → Return Visit (50% loss)
**Root cause**: No reminder, no incentive to return
**Fix**: Zalo Day-3 reminder with points balance
**Expected impact**: +15% return rate → +15 active members/month
**Cost**: 0₫ (Zalo is free)
