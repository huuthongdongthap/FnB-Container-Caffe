# ━━━ Wave 1: CSS Architecture & Cyberpunk Elements ━━━

Doi background cua .menu-item, .cart, .order-form thang mau translucent surface va them backdrop-filter: blur(12px). Dam bao co border suttle (1px solid rgba(255,255,255,0.1)) de ra chat kinh 3D. 1:Trien khai Cyberpunk / Neon Accents rieng cho Dark Mode. Mo css/styles.css. Tim block `[data-theme="dark"]`. Them bien CSS `--cyber-glow: 0 0 15px rgba(221, 189, 159, 0.4);` va `--cyber-gradient: linear-gradient(135deg, #FFD700, #FFA500, #FF4500);`. Appy glow effect (box-shadow) vao the button (nut bam primary) khi o dark mode. Apply text-shadow nhe cho cac .section-title. Verify hover co sang ruc len tren .menu-item. Luu lai.

2:Trien khai Advanced Micro-animations. Mo css/styles.css va js/ui-animations.js. Thiet lap animation "Ripple effect" hoac "Magnetic scale" cho `.cta-button`, `.order-btn`. Khi hover vao button: `transform: scale(1.03) translateY(-2px);` kem shadow sau. Doi voi .menu-item, them `transition: all 0.4s cubic-bezier(0.175, 0.885, 0.32, 1.275);`, hover vao thi `.menu-item-image` be-in (scale anh ben trong 1.1) kem brightness(110%). Luu CSS va JS.

3:Update Typography & Layout Polish. Mo css/styles.css. Chinh bo goc .menu-item, .order-form, .about-image thanh `--radius-xl` (24px) thay vi trung binh, de tao hinh dang organic/M3 mem mai hon. Them rules cho Scroll Reveal animation (opacity 0, transform translateY(30px), class .reveal.active -> opacity 1 translateY(0) transition 0.8s). Luu lai.
